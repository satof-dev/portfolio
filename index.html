    タイトル

    コミュパークあらた(架空サイト)

    目的・背景

    職業訓練の課題として架空の公共複合施設のWebサイトを制作しました

    施設の目的
    廃校をリニューアルして地域の公共複合施設とし、ランドマークとして地域の魅力を発信することで、多くの方にご利用いただくことを目的としています。 将来的には定住人口の増加や地域の持続的発展にもつなげることを意図しています

    Webサイトの目的
    Webサイトを中心にSNSと連携し、本施設が地域にとって有用な場であることを発信することを目的としています

    ターゲット

    メインターゲットは20代から30代の子育て中や子どもを望んでいる夫婦
    子どもたちが楽しく安心して遊べる場所や、子連れでも落ち着いて子育ての悩みや支援について相談できる場所を探している

    コンセプト

    親しみやすく、イベントを通じて活気があり、安心して過ごせることで「また訪れたい」と感じられるWebサイトです

    制作期間

    企画 2日
    フレームワーク 2日
    デザイン 5日
    コーディング 16日

    使用ツール

    Canva / Figma / WordPress / Visal Studio Code

    工夫したこと

    ・人々が集う場にふさわしい、温かみのあるデザインにしました ・開催イベントとSNS画像をトップページ直下に配置し、利用者の目に留まりやすくかつ参加意欲を高められるようにしました ・施設ページでは詳細情報や利用方法をわかりやすく整理し、安心して利用できるよう配慮しました ・お知らせ記事はWordPressから投稿可能とし、サイト運営側の利便性にも配慮しています
